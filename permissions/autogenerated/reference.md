## Default Permission

Default permissions for the plugin

#### This default permission set includes the following:

- `allow-create-temp-file`
- `allow-remove-temp-file`
- `allow-get-printers`
- `allow-get-printers-by-name`
- `allow-print-pdf`
- `allow-get-jobs`
- `allow-get-jobs-by-id`
- `allow-resume-job`
- `allow-restart-job`
- `allow-pause-job`
- `allow-remove-job`
